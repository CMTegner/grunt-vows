grunt-vows
==========
A grunt task for running your vows tests.

Installation
============
```
› npm install grunt-vows
```

Usage
=====
Add this line to your project's `grunt.js` gruntfile:

```
grunt.loadNpmTasks("grunt-vows");
```
Either run it via grunt:
```
› grunt vows
```
or as a stand-alone command:
```
› grunt-vows
```
